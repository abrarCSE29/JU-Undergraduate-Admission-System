<h1>This project implements the undergraduate admission system of Jahangirnagar University</h1>
<p>The project is developed using <em>JAVA</em></p>

<h3>Components</h3>
<ol>
  <li>JAVA - jdk-1.8.015</li>
  <li>IDE - Netbeans</li>
  <li>Database - MySQL (Using XAMPP)</li>
  <li>Should include the mysql-connector and jcalender jar files to the project while running it</li>
</ol>
